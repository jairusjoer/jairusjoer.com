## Headers and Text Styling

This document demonstrates **bold text**, *italic text*, and ***bold and italic text***.

You can also use *underscores* for *emphasis* and **strong text**.

Strikethrough is done with \~\~deleted text\~\~.

## Code Blocks and Inline Code

Inline code looks like this: `const example = "Hello, World!";`

Here's a Python code block:

```py
def greet(name):
    """A simple greeting function"""
    return f"Hello, {name}!"

# Multiple lines of code
users = ["Alice", "Bob", "Charlie"]
for user in users:
    print(greet(user))

```

And a JavaScript example:

```js
// Async function demonstration
async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error fetching data:', error);
    }
}

```
